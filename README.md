# Deepgram Go SDK (spec-mock)

Go SDK for the Deepgram speech-to-text platform: prerecorded
transcription, live streaming transcription, WebSocket and REST
transports, smart formatting, diarization, language detection,
entity detection, intents, sentiment, summarization, topic detection.
Runs on Go 1.22+. This is the prototype implementation generated by
the Deepgram Smithy spec pipeline; see [`REGEN.md`](REGEN.md) for the
pipeline and maintainer guide.

[![Go Reference](https://pkg.go.dev/badge/github.com/deepgram/spec-mock-go-sdk.svg)](https://pkg.go.dev/github.com/deepgram/spec-mock-go-sdk)

## Install

```bash
go get github.com/deepgram/spec-mock-go-sdk
```

## Quick start: prerecorded transcription

```go
package main

import (
	"context"
	"fmt"
	"log"

	rest "github.com/deepgram/spec-mock-go-sdk/pkg/client/listen/v1/rest"
)

func main() {
	client := rest.NewWithDefaults()
	resp, err := client.FromURL(context.Background(),
		"https://dpgr.am/spacewalk.wav",
		&rest.PreRecordedTranscriptionOptions{
			Model:       "nova-3",
			Punctuate:   true,
			SmartFormat: true,
		})
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(resp.Results.Channels[0].Alternatives[0].Transcript)
}
```

## Quick start: live streaming transcription

```go
package main

import (
	"context"
	"fmt"
	"log"

	ws "github.com/deepgram/spec-mock-go-sdk/pkg/client/listen/v1/websocket"
)

func main() {
	client := ws.NewWithDefaults()
	stream, err := client.Connect(context.Background(),
		&ws.LiveTranscriptionOptions{
			Model:          "nova-3",
			Encoding:       "linear16",
			SampleRate:     16000,
			InterimResults: true,
			SmartFormat:    true,
		})
	if err != nil {
		log.Fatal(err)
	}
	defer stream.Close()

	// Send audio chunks in a separate goroutine.
	go func() {
		audio := readAudioBytes() // your audio source
		_ = stream.SendAudio(audio)
		_ = stream.CloseStream()
	}()

	// Receive transcription events.
	for {
		event, err := stream.Recv()
		if err != nil {
			return
		}
		switch e := event.(type) {
		case *ws.ResultsEvent:
			if e.IsFinal {
				fmt.Println(e.Channel.Alternatives[0].Transcript)
			}
		case *ws.MetadataEvent:
			return // session ended
		case *ws.ErrorEvent:
			log.Fatal(e.Description)
		}
	}
}

func readAudioBytes() []byte { return nil }
```

Runnable examples for every public method:
- [`Client.Connect`](pkg/client/listen/v1/websocket/example_test.go) — open a streaming session.
- [`Stream.SendAudio`](pkg/client/listen/v1/websocket/example_test.go) — send raw audio bytes.
- [`Stream.Recv`](pkg/client/listen/v1/websocket/example_test.go) — receive the next event.
- [`Stream.Finalize`](pkg/client/listen/v1/websocket/example_test.go) — force a final result.
- [`Stream.KeepAlive`](pkg/client/listen/v1/websocket/example_test.go) — keep the session alive.
- [`Stream.CloseStream`](pkg/client/listen/v1/websocket/example_test.go) — graceful end-of-audio.
- [`LiveTranscriptionOptions`](pkg/client/listen/v1/websocket/example_test.go) — all 24 public query parameters.

Full godoc:

## Auth

Both clients pick up credentials from the environment:

```bash
export DEEPGRAM_API_KEY="your_api_key"
# or, for Bearer-token auth:
export DEEPGRAM_ACCESS_TOKEN="your_access_token"
```

`rest.NewWithDefaults()` and `ws.NewWithDefaults()` read these
automatically. `rest.New(apiKey, accessToken)` and
`ws.New(apiKey, accessToken)` take credentials explicitly. When both
are set, `accessToken` wins.

For custom configurations:

```go
client := rest.New("dg-key", "").
	WithBaseURL("https://api.example.com").
	WithHTTPClient(myHTTPClient)
```

## Test the examples

Every `Example_*` function in `pkg/.../example_test.go` is runnable
via `go test`:

```bash
go test ./pkg/...
```

Examples without an `// Output:` assertion are compile-checked.
Examples with one are also output-asserted. If a method signature
changes and an example stops compiling, the test suite fails.
